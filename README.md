## 📖 Sobre o Projeto

O **Projeto Final ADS** foi desenvolvido como trabalho de conclusão do curso de **Análise e Desenvolvimento de Sistemas**, tendo como objetivo aplicar na prática os conhecimentos adquiridos durante a graduação.

O sistema foi construído seguindo boas práticas de desenvolvimento, separando responsabilidades entre frontend e backend, utilizando tecnologias modernas para oferecer uma aplicação completa, intuitiva e escalável.

Além da implementação das funcionalidades principais, o projeto busca demonstrar conhecimentos em:

* Desenvolvimento Full Stack;
* Integração entre Frontend e Backend;
* Consumo e criação de APIs REST;
* Persistência de dados;
* Autenticação de usuários;
* Organização de código;
* Deploy em ambiente de produção.

# ⚙️ Como Executar Localmente

## Clone o repositório

```bash
git clone https://github.com/Cristantenicollas/Projeto_Final_ADS.git
```

Entre na pasta

```bash
cd Projeto_Final_ADS
```

Instale as dependências

```bash
npm install
```

Execute o projeto

```bash
npm run dev
```

Caso exista backend separado:

```bash
cd backend

npm install

npm start
```

# 📡 API

Exemplos de endpoints:

| Método | Endpoint      | Descrição        |
| ------ | ------------- | ---------------- |
| GET    | /             | Página inicial   |
| POST   | /login        | Login            |
| POST   | /register     | Cadastro         |
| GET    | /usuarios     | Lista usuários   |
| POST   | /usuarios     | Cria usuário     |
| PUT    | /usuarios/:id | Atualiza usuário |
| DELETE | /usuarios/:id | Remove usuário   |
